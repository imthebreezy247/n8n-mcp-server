QUICK START (15 MINUTES)
================================================================================

1. READ THE LEGAL DISCLAIMER
   → File: docs/LEGAL.md
   → IMPORTANT: Using copyrighted content without permission is ILLEGAL

2. GET TMDB API KEY (Free, 2 minutes)
   → Visit: https://www.themoviedb.org/signup
   → Go to Settings → API → Request API Key
   → Choose "Developer" option

3. CONFIGURE ENVIRONMENT
   → cp config/.env.example config/.env
   → Edit config/.env
   → Add your TMDB_API_KEY
   → Set DRY_RUN_MODE=true

4. INSTALL DEPENDENCIES
   → cd scripts
   → npm install

5. START VIDEO PROCESSOR
   → node video-processor.js
   → Keep this terminal running

6. START N8N (new terminal)
   → export TMDB_API_KEY=your_key_here
   → export DRY_RUN_MODE=true
   → n8n

7. IMPORT WORKFLOW
   → Open http://localhost:5678 in browser
   → Click "Workflows" → "Import Workflow"
   → Upload: workflows/tiktok-auto-clipper-workflow.json

8. TEST IT
   → Open imported workflow
   → Click "Execute Workflow"
   → Watch it run!
